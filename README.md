# Hackly - Frontend

Hackly is a Hackathon Management Platform that allows organizers to create hackathons, manage registrations, collect payments, and track participants.

## Live Demo

https://hackathon-hub-frontend.vercel.app

---

## Features

- Google OAuth Authentication
- Browse Upcoming Hackathons
- Event Registration
- Ticket Booking
- Payment Integration (Razorpay)
- Event Dashboard
- Registration Forms
- Participant Verification
- Protected Routes
- Responsive UI

---

## Tech Stack

### Frontend

- React.js
- React Router DOM
- Axios
- Context API
- CSS
